# Ultra.cc Scripts

Scripts specially designed to be used on Ultra.cc services. For more information on the scripts, visit the [documentation](https://docs.usbx.me/) or the [Ultra.cc Community Discord Server](https://discord.com/invite/YXGYS9D).

The scripts here are provided as-is and Ultra.cc will not provide any official support for said scripts. Should you want to use it on your own setup, you may have to edit it accordingly.
